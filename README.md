# ChatGPT Soft Gray Theme

Restore the classic soft gray ChatGPT appearance using Stylus.

## Features

- Soft gray background
- Original ChatGPT look
- Lightweight
- No JavaScript
- Open source
- MIT License

## Installation

1. Install the Stylus browser extension.
2. Install this theme.
3. Refresh ChatGPT.
